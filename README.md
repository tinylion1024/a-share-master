# A股散户盈利助手

> 帮散户赚钱，以「复利增长」为北极星目标

---

## 核心理念

散户亏损的根源不是选股能力，而是**心态**、**认知**和**执行**。

本Skill不追求单次暴利，而是通过：
- **标准化流程** — 消除情绪干扰
- **四维评级** — 让选股有据可依
- **反馈循环** — 持续进化能力

---

## 核心体系

### 三维看市场

| 维度 | 内容 |
|------|------|
| 消息 | 政策、宏观、行业、公司公告 |
| 情绪 | 散户情绪、大V观点、游资动向 |
| 技术 | 趋势、均线、成交量、MACD |

### 四维选股评级

| 维度 | 说明 |
|------|------|
| 机会⭐ | 上涨空间与概率 |
| 安全⭐ | 风险程度（R3红线） |
| 确定⭐ | 上涨逻辑清晰度 |
| 舒适⭐ | 持有体验（位置/波动） |

## 15个核心场景

**赚钱引擎**（买/卖/持）

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 选股 | 什么能赚钱 | 机会清单 + 理由 |
| 诊股 | 能买XXX吗 | 买/观望/不买 + 价位 |
| 持仓诊断 | 要不要卖 | 持有/加/减/清 |
| 解套 | 被套了 | 补仓/割肉/换股 |
| 止损 | 亏多少走 | 止损价 + 纪律 |
| 止盈 | 赚多少跑 | 目标价 + 分批 |

**学习引擎**（市场/反思）

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 大盘前瞻 | 今天怎么看 | 方向 + 仓位 |
| 大盘总结 | 为什么涨/跌 | 归因 + 规律 |
| 每日复盘 | 操作对吗 | 对错 + 改进 |
| 案例拆解 | 为什么涨 | 深层逻辑 |
| 错误反思 | 为什么亏 | 归因 + 改进 |
| 经验沉淀 | 怎么赚到的 | 成功逻辑 |

**进化引擎**（积累/优化）

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 案例记录 | 记录这笔 | 归档案例库 |
| 规律更新 | 更新规则 | 规则优化 |
| 模型优化 | 调整参数 | 参数/权重 |

python3 scripts/post_market.py

## 硬规则

违反以下任一规则，直接取消资格：

| 规则 | 说明 |
|------|------|
| R3红线 | 安全⭐<3 禁止推荐 |
| 止损7% | 不执行止损是亏损主因 |
| T+1 | 当日买不可卖 |
| 流动性 | 成交额<5000万不做主要标的 |
| 仓位 | 单只≤30%，总≤50% |

## 适用场景

**触发本Skill时：**

- A股分析、选股、诊股、买股、卖股、持仓
- 盘前、盘中、盘后、复盘
- 涨停板、龙头股、连板股、高位股
- 风险扫描、止损、止盈、解套
- 交易计划、仓位管理、主力资金

**不适用：**

- 美股、港股、期货、期权、数字货币
- 基本面长线投资、量化策略开发
