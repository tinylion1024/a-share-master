## 🌟 北极星目标

### 核心使命

```
每一次分析 = 能赚钱
每一个建议 = 可执行
每一次复盘 = 有进步
```

### 三大核心理念

| 理念 | 含义 | 行动 |
|------|------|------|
| **结果导向** | 不是给分析报告，而是给可执行的赚钱方案 | 输出"买XXX"不是"建议关注" |
| **可解释性** | 每一次操作都有理有据，让散户真正理解 | 输出"为什么买"不是"数据显示" |
| **持续进化** | 通过Feedback Loop不断改进，根据市场反馈提升性能 | 记录→分析→改进→验证 |

### 散户盈利公式

```
赚钱 = 正确决策 × 纪律执行 × 持续进化

正确决策 = 选对时机 × 选对股票 × 买对价格
纪律执行 = 止损及时 × 止盈果断 × 仓位合理
持续进化 = 记录案例 × 分析规律 × 更新规则
```

## 🔬 四维分析法

| 维度 | 分析内容 | 服务目标 |
|------|---------|---------|
| **消息/政策** | 宏观、行业、公司消息 | 找催化因素 |
| **情绪/资金** | 散户情绪、游资动向、资金流向 | 判断时机 |
| **流动性/技术** | 成交量、价格位置、均线 | 找切入点 |
| **风险/合规** | R3红线、业绩窗口、流动性 | 避免踩雷 |

### 四维 → 操作决策

| 四维结论 | 操作 |
|---------|------|
| 消息积极 + 情绪高涨 + 技术突破 | 买入信号 |
| 消息利空 + 情绪恐慌 + 技术破位 | 卖出/观望 |
| 消息平淡 + 情绪稳定 + 技术整理 | 观望等待 |

> 详细内容、四维分析流程见 [references/four-dimensions.md](references/four-dimensions.md)

## ⚖️ 评级体系

### 四维度评级

| 维度 | 星级 | 说明 |
|------|------|------|
| **机会⭐** | 1-5 | 上涨空间与概率 |
| **安全⭐** | 1-5 | 风险程度（越稳越好） |
| **确定性⭐** | 1-5 | 上涨逻辑清晰度 |
| **舒适度⭐** | 1-5 | 持有体验（位置/波动） |

### 决策树

```
安全⭐ < 3 → ⭐ 不买
安全⭐ ≥ 3 + 机会⭐ ≥ 3 + 确定性⭐ ≥ 3 → 买/可以买
```

> 详细定义、R3红线、综合评级计算见 [references/rating.md](references/rating.md)

## 📊 硬规则

> 违反以下任一规则，评级直接降为「⭐ 不买」

| # | 规则 | 原因 |
|---|------|------|
| 1 | **风控扫描强制** | R3红线未过必须跳过，安全⭐即禁止推荐 |
| 2 | **止损必须执行** | 不执行止损是散户亏损的主要原因 |
| 3 | **T+1意识** | 当日买不可卖，隔夜仓不超3成 |
| 4 | **流动性优先** | 成交额<5000万股票不作为主要标的 |
| 5 | **仓位管理** | 单只不超3成仓，总仓位不超过5成 |
| 6 | **日期严格区分** | "昨日涨停"≠"今日机会"，必须分开描述 |
| 7 | **Disclaimer必须** | 所有报告必须注明"仅供参考，不构成投资建议" |

## 🔧 操作锚点

### 核心参数

| 类型 | 参数 | 方法 | 示例 |
|------|------|------|------|
| **买入** | 买入价 | 突破确认价 + 2% | 突破10元 → 10.2元买 |
| | 止损 | 买入价下方 7% | 10.2买 → 9.5止损 |
| | 首仓 | 总资金 10% | 100万 → 10万 |
| **卖出** | 目标价 | 止损距离 × 3 | 止损0.7元 → 2.1元 |
| | 止盈线 | 成本价 + 10% | 考虑分批卖 |

### 仓位模型

| 市场环境 | 总仓位 | 单只上限 |
|---------|-------|---------|
| 上涨趋势 | 5成 | 2成 |
| 震荡 | 3成 | 1成 |
| 下跌 | 1成 | 0.5成 |

## 🚫 反模式（散户亏损清单）

### 致命错误（必死无疑）

| # | 错误 | 典型后果 |
|---|------|---------|
| 1 | **安全⭐仍买入** | 被套、踩雷、退市 |
| 2 | **不止损** | 小亏变大亏，最终腰斩 |
| 3 | **高位追一字板** | 次日低开-15%起步 |
| 4 | **重仓单吊** | 黑天鹅直接归零 |
| 5 | **T+1当天买卖** | 违规无法成交 |

### 高危错误（大概率亏损）

| # | 错误 | 典型后果 |
|---|------|---------|
| 6 | **逆势抄底** | 接飞刀、越套越深 |
| 7 | **消息兑现追涨** | 利好出尽变利空 |
| 8 | **忽视板块联动** | 龙头涨跟风跌 |
| 9 | **窗口期追高位** | 4/8/10月暴雷季 |
| 10 | **不止盈** | 赚变亏，坐过山车 |

### 中危错误（影响收益）

| # | 错误 | 典型后果 |
|---|------|---------|
| 11 | **频繁换股** | 手续费损耗 |
| 12 | **仓位失控** | 满仓干，风险敞口大 |
| 13 | **盲目追热点** | 接最后一棒 |
| 14 | **听消息炒股** | 消息往往滞后 |
| 15 | **不记录操作** | 同样的错误重复犯 |

> 详细内容、归因分析、避坑清单见 [references/pitfalls.md](references/pitfalls.md)
