# 大盘前瞻

> 预判今日市场方向和操作策略

## 触发词

- "今天怎么看"
- "大盘今天怎么走"
- "今天能买吗"
- "市场展望"

## 前置确认

- [ ] 确认时间（早盘/午盘）
- [ ] 确认隔夜信息（美股/A50/政策）

## 分析步骤

### Step 1：隔夜信息收集

| 信息源 | 内容 | 影响 |
|--------|------|------|
| 美股 | 涨跌、热点 | 情绪传导 |
| A50 | 期货表现 | 开盘参考 |
| 政策 | 利好/利空 | 方向指引 |
| 外盘商品 | 原油/黄金 | 板块影响 |

### Step 2：三维信号评估

| 维度 | 分析要点 |
|------|----------|
| 消息 | 政策导向、宏观数据 |
| 情绪 | TGB热度、V观点 |
| 技术 | 指数位置、趋势、成交量 |

### Step 3：信号组合判断

| 信号 | 市场判断 | 操作建议 |
|------|----------|----------|
| 🟢🟢🟢 | 强势上涨 | 积极做多 |
| 🟢🟡🟢 | 震荡偏强 | 精选个股 |
| 🟡🟡🟡 | 震荡整理 | 高抛低吸 |
| 🔴🔴🔴 | 弱势下跌 | 休息/空仓 |

```markdown
## 【今日大盘前瞻】

**大盘判断**：强势/震荡偏强/震荡/弱势
**三维信号**：消息🟢 情绪🟢 技术🟢

【大盘环境】
上证：XXX点  创业板：XXX点
成交额预期：XXX亿

【情绪信号】
- TGB热度：XXX
- V观点：做多/观望/谨慎
- 赚钱效应：强/中/弱

【操作策略】
仓位建议：X成
重点关注：XXX板块/XXX股
风险提示：XXX

【双场景预案】
乐观条件：XXX → 应对：XXX
悲观条件：XXX → 应对：XXX
```

## 硬规则检查

- ❌ 大盘弱势 → 控制仓位
- ❌ 情绪低迷 → 不追涨
