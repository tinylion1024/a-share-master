# 选股

> 筛选有机会的股票，给出理由和预期收益

## 触发词

- "有什么能赚钱"
- "推荐股票"
- "有什么好股"
- "帮我选股"

## 前置确认

- [ ] 确认持仓周期（短线/中线/长线）
- [ ] 确认风险偏好（保守/平衡/激进）
- [ ] 确认仓位上限

## 分析步骤

### Step 1：三维看市场

判断大盘环境是否适合选股：

| 信号 | 环境 | 操作 |
|------|------|------|
| 🟢🟢🟢 | 强势 | 积极选股 |
| 🟢🟡🟢 | 震荡偏强 | 精选个股 |
| 🟡🟡🟡 | 震荡 | 控制仓位 |
| 🔴🔴🔴 | 弱势 | 休息/空仓 |

### Step 2：筛选条件

按优先级筛选：

| 优先级 | 条件 | 说明 |
|--------|------|------|
| 1 | 成交额≥5000万 | 流动性保障 |
| 2 | 趋势向上 | MA多头排列 |
| 3 | R3安全⭐≥3 | 硬规则过滤 |
| 4 | 有催化剂 | 政策/业绩/事件 |

### Step 3：四维评级

对候选股进行四维评级：

```
综合分 = 机会⭐×30% + 安全⭐×25% + 确定⭐×25% + 舒适⭐×20%

4.5+ → ⭐⭐⭐⭐⭐ 立即买
3.5+ → ⭐⭐⭐⭐ 可以买
2.5+ → ⭐⭐⭐ 观望
```

## 输出规范

```markdown
【选股报告】
大盘环境：XXX
推荐标的：
1. [股票名称] 代码：XXX
   - 四维：机会X 安全X 确定X 舒适X
   - 综合分：X.X
   - 推荐理由：XXX
   - 买入区间：XX-XX
   - 止损价：XX（-7%）
   - 目标价：XX（+X%）
   - 仓位建议：X成
```

## 硬规则检查

- ❌ 安全⭐<3 → 不推荐
- ❌ 成交额<5000万 → 不推荐
- ❌ 弱势大盘 → 休息
