# 止损

> 亏损多少必须走

## 触发词

- "亏多少必须走"
- "止损设多少"
- "XXX跌了要不要卖"
- "亏了X%怎么办"

## 前置确认

- [ ] 确认股票代码
- [ ] 确认买入成本价
- [ ] 确认当前亏损幅度

## 止损原则

### 硬规则：亏损7%必须走

| 亏损幅度 | 操作 | 理由 |
|----------|------|------|
| -3% | 警告 | 观察 |
| -5% | 关注 | 可能破止损 |
| -7% | **必须走** | 硬规则，不侥幸 |
| >-7% | 绝不留 | 防止深套 |

### 止损方式

| 方式 | 适用场景 | 操作 |
|------|----------|------|
| 硬止损 | 所有股票 | -7%无条件走 |
| 移动止损 | 盈利后 | 从成本价跟踪 |
| 时间止损 | 短线 | 5日不涨走 |

## 分析步骤

### Step 1：计算止损价

```
止损价 = 买入价 × (1 - 7%)
```

### Step 2：检查是否触及

| 当前价位 | 状态 | 操作 |
|----------|------|------|
| >成本价 | 盈利 | 持有，移动止损 |
| 成本价~-5% | 正常回撤 | 观察 |
| -5%~-7% | 警告区 | 准备执行 |
| ≤-7% | **触发止损** | 立即执行 |

### Step 3：执行决策

```
已盈利 → 移动止损（保本）
已亏损 → -7%硬止损
```

## 输出规范

```markdown
## [止损分析]
股票：XXX 代码：XXX
买入价：XX.X  现价：XX.X  亏损：-X%

**止损执行**：是/否

若执行：
- 卖出数量：全部/减半
- 预计亏损：XXX元

若不执行（仅盈利时）：
- 移动止损价：XX
```

## 硬规则检查

- ❌ 亏损≥7% → 必须走，不侥幸
- ❌ 下跌趋势 → 不抄底
- ❌ 不补仓摊成本 → 越套越深
