# 案例记录

> 记录操作案例到案例库，持续积累

## 触发词

- "记录这笔"
- "存个案例"
- "记一下这笔操作"
- "归档这笔交易"

## 前置确认

- [ ] 确认股票代码
- [ ] 确认买入卖出价位
- [ ] 确认盈亏结果
- [ ] 确认操作类型（买入/卖出/持有）

## 案例要素

| 要素 | 内容 |
|------|------|
| 股票 | 名称/代码 |
| 时间 | 买入/卖出日期 |
| 价位 | 成本/卖出价 |
| 盈亏 | 金额/百分比 |
| 仓位 | 占比 |
| 类型 | 短线/中线/长线 |
| 操作理由 | 为什么买/卖 |
| 关键得失 | 做对/做错 |

## 分析步骤

### Step 1：收集信息

```
股票代码：XXX
买入日期：XXXX-XX-XX
买入价位：XX.XX
卖出日期：XXXX-XX-XX（未出填持有）
卖出价位：XX.XX（未出填现价）
持仓数量：XXX股
操作类型：买入/卖出/持有
```

### Step 2：评估结果

```
盈亏金额：XXX元
盈亏比例：+X%/-X%
是否符合预期：XXX
```

### Step 3：记录归档

```
案例库路径：references/cases/YYYY/MM/
文件名：YYYYMMDD_股票代码_盈亏.md
```
