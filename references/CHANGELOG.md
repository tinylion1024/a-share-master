# 更新日志

## v1.0.0 (2024-04-25)

**初始版本**

整合以下 7 个原有 skill：
- `a-share-daily-pre-opening-report` - 每日盘前报告
- `a-share-detailed-reviewer` - 详细复盘工具
- `a-share-integrated-expert` - 综合专家
- `a-share-market-synthesis` - 市场综合分析
- `a-share-v2-analyzer` - V2 分析器
- `high-pr-stock-picker` - 高性价比股票筛选
- `trading-plan-generator` - 交易计划生成器

**核心功能**：
- 四维分析法（新闻/政策、情绪/资金、流动性/技术、风险/合规）
- 双引擎驱动（MX 数据 + TGB 情绪）
- 六大功能模块
- 四级评级体系（R/O/C/舒适度）
