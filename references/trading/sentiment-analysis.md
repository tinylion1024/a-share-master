# 情绪分析方法

## 情绪周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| **启动期** | 龙头出现、连板、赚钱效应扩散 | 积极参与 |
| **繁荣期** | 全场亢奋、消息满天飞、菜市场都在讨论 | 谨慎追高 |
| **混沌期** | 板块轮动快、没有明确主线 | 观望为主 |
| **退潮期** | 龙头炸板、跌停扩散、亏钱效应出现 | 空仓休息 |

## 情绪指标

### 散户情绪
- TGB 热门话题关键词
- V 共识方向（看多 vs 看空）
- 评论区情绪（恐慌/亢奋）

### 资金情绪
- 主力净流入 vs 散户净流出
- 涨停家数（>50 为过热）
- 跌停家数（>20 为冰点）

### 赚钱/亏钱效应
- 昨日涨停今日表现（平均溢价）
- 昨日炸板股今日修复情况

## 背离信号

| 背离类型 | 信号含义 |
|----------|----------|
| 价格↑ + 主力净流出↓ | 可能是诱多 |
| 价格↓ + 主力净流入↑ | 可能是诱空 |
| 指数↑ + 成交量↓ | 虚假繁荣 |
| 涨停↑ + 炸板率↑ | 情绪分歧 |
