# 行业板块分类

## 主要板块

| 板块 | 包含行业 | 代表股票 |
|------|----------|----------|
| 科技 | 半导体、软件、电子 | 宁德时代、中芯国际 |
| 新能源 | 锂电池、光伏、风电 | 比亚迪、隆基绿能 |
| 消费 | 白酒、食品饮料、家电 | 贵州茅台、美的集团 |
| 医药 | 生物医药、医疗器械、中药 | 恒瑞医药、迈瑞医疗 |
| 金融 | 银行、保险、证券 | 招商银行、中国平安 |
| 房地产 | 房地产开发、建材 | 万科A、保利发展 |
| 军工 | 航空航天、兵器装备 | 中航沈飞、航发动力 |
| 化工 | 基础化工、新材料 | 万华化学、卫星化学 |

## 板块轮动规律

1. **政策驱动**：消息面刺激相关板块短期爆发
2. **业绩驱动**：财报季，业绩超预期板块表现
3. **资金驱动**：主力资金持续流入板块
4. **防御切换**：市场不好时，资金切换到消费、医药
