# 安全与风险管理

## 风险等级体系

| 等级 | 风险程度 | 建议 |
|------|----------|------|
| **R1** | 低风险 | 可正常关注 |
| **R2** | 中风险 | 谨慎参与，控制仓位 |
| **R3** | 高风险 | 禁止推荐，回避 |

## R3 红线项目（绝对禁止）

以下情况必须将风险等级标记为 **R3**：

1. **立案调查** - 上市公司被证监会立案调查
2. **退市风险** - *ST、**ST 股票
3. **大比例减持** - 大股东或高管拟减持 > 5%
4. **业绩巨亏** - 季度/年报亏损超预期
5. **业绩窗口期高位股** - 高位股在窗口期默认 R3

## 业绩窗口期

| 时间窗口 | 风险处置 |
|---------|---------|
| 4月20日-30日 | 年报/一季报窗口，高位股默认 R3/C3 |
| 8月20日-31日 | 中报窗口 |
| 10月20日-31日 | 三季报窗口 |

## 仓位管理原则

| 风险等级 | 最大仓位 |
|----------|----------|
| R1 | 30% |
| R2 | 15% |
| R3 | 0%（禁止） |

## 止损原则

- **固定止损**：亏损达到 7%-10% 必须止损
- **移动止损**：盈利超过 15% 后，止损线上移至成本价
- **时间止损**：持仓超过 5 个交易日无明显动静，考虑出局

## 数据安全

- API 密钥存储在环境变量中，不写入代码
- 敏感数据不上传到任何外部系统
- 定期更换 API 密钥
