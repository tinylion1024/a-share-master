# 交易计划 (Trading Plan Generator)

## 工作流程

1. **数据采集**：使用 `mx-finance-search` 或 `mx-data` 获取上一交易日指标（成交量、资金流向、涨跌）
2. **情绪分析**：使用 `taoguba-hot` 获取 15:00 后 V 观点
3. **场景映射**
   - **乐观**：定义触发条件（如龙头撬动、成交量阈值）和目标板块
   - **悲观**：定义风险信号（如跌停扩散）和防御动作
4. **可执行锚点**：指定具体价位、成交量目标、止损条件

## 坑点

- 避免泛泛建议；使用具体股票代码作为"锚点"（如当前龙头或"龙"股）
- 确保"风险控制"部分是不可妥协的，识别具体的"突破"或"破位"点
- 运行前验证是否为交易日
